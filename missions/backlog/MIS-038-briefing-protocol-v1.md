# MIS-038 — Diseñar Briefing Protocol v1.0

**Area:** CAO · **Gremio:** Centinelas · **Tipo:** 🤖 Digital · **Prioridad:** 🔴 Crítica · **Esfuerzo:** M

## Historia

Como coordinador CAO, quiero un protocolo estándar de briefing para activar agentes, para que cada sesión comience con contexto suficiente.

## Criterios de aceptación

- [ ] Documentado en protocols/briefing-v1.md
- [ ] Define: estructura mínima, archivos a leer, orden de inicialización
- [ ] Plantilla con variables reemplazables
- [ ] Probado en sesión real con tiempo < 3 minutos

## Valor epistémico

Hace explícito qué información es realmente crítica para que un agente funcione.

## Valor pragmático

Reduce tiempo muerto al inicio de cada sesión.
