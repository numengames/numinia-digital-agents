## 📖 Historia

Como Oráculo de Numen Games, quiero poder hablar directamente con Nimrod (y el resto de agentes) vía Telegram, para participar en la CAO sin necesitar acceso técnico al servidor.

## ✅ Criterios de aceptación

- [ ] Primer Oráculo (@Wolfstein_Wagen) añadido como sender autorizado
- [ ] El Oráculo puede enviar mensajes a Nimrod y recibir respuestas
- [ ] Protocolo definido: cómo se añaden nuevos Oráculos
- [ ] Todos los Oráculos activos con acceso (María, Christian, Clio, Dani)
- [ ] Documento de protocolo de acceso creado

## 🧠 Valor epistémico

Valida si el sistema CAO puede operar con múltiples humanos interactuando con el mismo agente, y cómo gestionar el contexto compartido vs. privado.

## ⚡ Valor pragmático

Los Oráculos pueden delegar tareas a Nimrod directamente, sin depender de Pablo como intermediario. La CAO escala.

## 📋 Protocolo de onboarding de Oráculo

1. El Oráculo envía cualquier mensaje al bot de Telegram de Nimrod
2. Nimrod captura su Telegram ID del metadata
3. Pablo aprueba añadir el ID en el config de OpenClaw
4. El ID se añade a `authorizedSenders` en el config del servidor
5. El Oráculo ya puede hablar con Nimrod
