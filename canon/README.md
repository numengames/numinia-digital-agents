# Seminal Documents

This folder contains the canonical source documents of the Numinia / Numen Games universe. They are the epistemic foundation — the single source of truth for lore, philosophy, brand, organization, and game design.

**These documents must not be modified.** They are reference material. When the original source is updated, the file here is replaced wholesale.

## Purpose

These documents serve two functions:

- **Epistemic value** — Coherence of meaning. They define what Numinia and Numen Games are, how they relate, and what principles guide them.
- **Pragmatic value** — Efficacy in action. They enable consistent decisions in design, development, brand, and culture.

For AI agents: **read these documents before answering questions about Numinia lore, brand, philosophy, or organizational model.** The other documentation files (README, CLAUDE.md, docs/) contain summaries; these are the full canonical sources.
