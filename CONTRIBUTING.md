# Contributing to the Archive Summa

This repository is the canonical truth of the Numinia Digital Agents system.

## Who can contribute

| Role | Can create | Can modify | PR approved by |
|------|-----------|-----------|----------------|
| Oracle | Everything | Everything | Oracle |
| Active agent | Own missions, protocols | Own STATUS.md | Oracle |
| External contributor | Proposals only | — | Oracle |

## Rules

1. **Never modify `canon/`** — these documents are immutable by policy
2. **Never include real credentials, IPs, or tokens** — use credential-map.md for structure
3. **Commit messages in English** — always
4. **One PR per logical change** — don't bundle unrelated modifications
5. **Missions follow the TEMPLATE.md format** — PRs rejected otherwise

## Pull Request process

1. Fork or branch from `main`
2. Create your changes
3. Open PR with descriptive title and body
4. Wait for Oracle approval
5. Oracle merges — you don't merge your own PRs

## For agents

Read `protocols/P-001-briefing-agente-v1.md` before starting any work session.
